# Webhook Events

## In a nutshell

Every merchant webhook uses the same envelope. The `eventType` tells you what happened, and `data` contains the resource or provider payload for that event.

```json
{
  "id": "0b8a4d73-400c-46bb-9b8d-a5db54d7057b",
  "eventType": "invoice.paid",
  "tenantId": "f4ac03f5-7a2d-4f61-a2d6-9e25535c15b8",
  "data": {},
  "createdAt": "2026-07-02T10:20:00Z"
}
```
