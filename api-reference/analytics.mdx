# Analytics API

## Planned

Analytics endpoints are planned but not currently exposed as public API.

Planned metrics may include:

- MRR
- ARR
- Gross revenue
- Net revenue
- Active subscriptions
- Failed payment rate
- Revenue by plan
- Upcoming renewals

Until analytics endpoints are available, use invoice, subscription, and webhook data in your own system for reporting.
